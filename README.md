# Ayudantías

Repositorio personal para organizar el material que preparo, adapto o utilizo en ayudantías universitarias.

Actualmente este repositorio reúne material principalmente de dos cursos:

- **Campos y Ondas**
- **Física Matemática 2**

La idea es ir guardando aquí notebooks pedagógicos, guías resueltas, apuntes, desarrollos en \LaTeX, pautas para pizarra y material de apoyo que pueda reutilizarse, corregirse y ampliarse con el tiempo.
